In this project, I used Tirthraj Dharane's model and tested it. It uses two classes — safe driver and unsafe driver. I added a green light for a safe driver and a red light for an unsafe driver, and also added an on-screen alert to indicate whether the driver is safe or unsafe. The model is performing well on both images and in cars. I tested it on my laptop using images, while Tirthraj Dharane tested it in his car.
